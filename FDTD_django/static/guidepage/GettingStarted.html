REST API and this guide are a WORK IN PROGRESS and APIs are subject to change. It is intended for early adopters who want to test out the REST API and provide feedback

Getting Started

The REST API lets you interact with Refractive Index Database from anything that can send an HTTP request. There are many things you can do with our REST API. For example:

  • Application written in any programming language can interact with data on our Refractive Index Database.
  • You can Search any element easily with our REST API

Quick Reference

All API access is over HTTP, and accessed via the http://www.infomagnetic.com domain.

query

URL HTTP Verb Functionality
/elementlistitems/<elementname>/ GET Retrieving paper list according to element
/elementlistitemsdetail/<pk>/ GET Retrieving element detail information according to primary key

Response Format

The response format for all requests is a JSON object. For example, /elementlistitemsdetail/14/ will return the JSON object:

{ "DATA": [ { "type": "formula 4", "range": "0.59 4.6", "coefficients": "6.346 0.342 0 0.09 1 0 0 0 1 -0.0011 2" } ], "REFERENCES": "K. F. Hulme et al. Synthetic proustite (Ag3AsS3)", "COMMENTS": "Extraordinary ray (e), 20 C", "ELEMENT": "Ag3AsS3", "PAPER": "Hulme-e" }
